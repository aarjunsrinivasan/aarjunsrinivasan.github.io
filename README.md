# Personal Portfolio

## Project structure

- **`index.html`** — Page structure and content (nav, hero, stats, projects, experience, publications, blogs, contact, footer).
- **`css/style.css`** — All styles: theme variables, layout, components, responsive rules.
- **`js/main.js`** — Theme toggle, mobile menu, smooth scroll, scroll effects, fade-in, nav highlighting, stats counter.
- **`blog/`** — Blog posts (e.g. `pca-svd.html`).
